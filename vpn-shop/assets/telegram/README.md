# Telegram Media Assets

Put generated SilentConnect images here:

```text
avatar.png
welcome.png
quickstart.png
```

`avatar.png` is uploaded through BotFather. `welcome.png` and `quickstart.png` can be referenced from `.env.silentconnect` as:

```text
WELCOME_MEDIA=assets/telegram/welcome.png
QUICKSTART_MEDIA=assets/telegram/quickstart.png
```
